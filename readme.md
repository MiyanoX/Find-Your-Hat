# Hat Find Game

## Gettinng start

Package prompt-sync is required. Run below command to install this package:

```bash
npn i prompt-sync
```

Run below command to start game:

```bash
node main.js
```

## Tutorial

### Input

- u -> up
- d -> down
- l -> left
- r -> right

### Icon:

- ^ -> Hat
- O -> Hole
- â–‘ -> Field
- \*  -> Current Place
