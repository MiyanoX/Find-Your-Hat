## Gettinng start

Package prompt-sync is required. Run below command to install this package:

```bash
npm i prompt-sync
npm i terminal-kit
```

Run below command to start game:

```bash
node main.js
```

## Tutorial

### Input

- w -> up
- s -> down
- a -> left
- d -> right

### Icon

- \_A_ -> Hat
- ( ) -> Hole
- ‘   ’ -> Field
- <\*>  -> Current Place
